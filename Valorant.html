Como melhorar o FPS no Valorant

Comando usado:

Windows + R

Passo a Passo:

Acesse o seguinte caminho:

Windows + R → Local → Valorant → Saved → Config → GameUserSettings

Resoluções recomendadas:

  • Se tiver uma que gosta pode utilizar
  • 1920x1080
  • 1720x1080
  • 1600x1200
  • 1600x900
  • 1400x1050
  • 1280x1024

Configurações do GameUserSettings.ini

bShowFPS=True
FrontendFrameRateLimit=120 → Coloque o valor desejado
FrameRateLimit=144 → Coloque o valor desejado

Mudar a resolução do Valorant:

Os quatro abaixo são os locais para alterar as resoluções, mas nos dois últimos, use o mesmo número dos de cima.

ResolutionSizeX=800 → Escolha uma resolução recomendada
ResolutionSizeY=600 → Escolha uma resolução recomendada
LastUserConfirmedResolutionSizeX=1280 → Escolha uma resolução recomendada
LastUserConfirmedResolutionSizeY=720 → Escolha uma resolução recomendada
DesiredScreenWidth=1280 → Mesmo valor que o de cima.
DesiredScreenHeight=720 → Mesmo valor que o de cima.
LastUserConfirmedDesiredScreenWidth=1280 → Mesmo valor que o de cima.
LastUserConfirmedDesiredScreenHeight=720 → Mesmo valor que o de cima.

Outras configurações:

[ScalabilityGroups]
sg.ResolutionQuality=75 → Recomenda-se 80, 70 ou 0
sg.ViewDistanceQuality=0
sg.AntiAliasingQuality=0
sg.ShadowQuality=0
sg.PostProcessQuality=0
sg.TextureQuality=0
sg.EffectsQuality=0
sg.FoliageQuality=0
sg.ShadingQuality=0

Modo de tela:

LastConfirmedFullscreenMode=0
PreferredFullscreenMode=1

*Aviso*: Após fazer todas as alterações, vá ao arquivo GameUserSettings.ini, clique com o botão direito e selecione "Somente leitura".
